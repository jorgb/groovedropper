Keyboard Controls
Playback
SPACEPlay / Pause
SHIFT + SPACERestart from start offset
CTRL + SPACESet start offset to playhead position
Navigation
HOMEJump to absolute start (offset 0)
← / →Prev / next marker
SHIFT + ←Prev marker (fast step)
LEFT MOUSEPick playhead position in waveform
Markers
M / SHIFT + MToggle marker at start offset
CTRL + ENTERPin marker at current playhead
DDelete all markers
L / SHIFT + LAuto set linear / random markers
Samples
R / SHIFT + RRandom pick next sample / randomize offset in sample
T / SHIFT + TFind transient (soft or SHIFT for harder edges)
AArchive sample (mutable only)
XCut / Merge sample — opens dialog (mutable only)
Pitch
, / .−1 / +1 semitone
SHIFT + , / SHIFT + .−10 / +10 cents
/Reset pitch
Pick Window
JSelect older sample in current pick window
KSelect newer sample in current pick window
SHIFT + JJump to first (oldest) sample in pick window
SHIFT + KJump to last (most recently picked) sample in pick window
Quick Pick
VSave to next free slot
1–9, 0Recall slot (0 = slot 10)
SHIFT + 1–9, 0Save to slot
Export / Saving
SDownload current slice till end (or marker boundaries)
SHIFT + SDownload original sample (and sliced markers as zip)
Other
CTRL + UCopy link to clipboard
F2Rename sample (mutable only)
?Show / hide this dialog
