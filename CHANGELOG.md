# Changelog

## v0.9

### Added
- Initial public release
- WAV sample browser with waveform display and playhead
- Random and sequential sample navigation
- Label and preset system with OR/AND filtering
- Scan folder management with per-folder auto-labels
- Offset/start-point selection with drag and keyboard nudge
- Pitch adjustment (semitones + cents) with drag controls
- Three themes: Default, Hacker, ProTracker
- Loop toggle and audio slice export
- Ignore list (.grvignore) support
- First-run guide screen
- Electron installer (Windows, macOS, Linux) and plain Python runner
- Database migration system for future schema upgrades
